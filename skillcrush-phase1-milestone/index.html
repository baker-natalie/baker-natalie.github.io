What I've Learned

I have been studying web development seriously for over two years, and I’ve worked full time as a developer for over a year. So I came to this course expecting it to be a breeze, but it was a great refresher on best practices and how to build a website with HTML & CSS only- NO JavaScript! Every web developer should have the CSS chops to build a great website from scratch, and this course really gets you exercising them! I will be able to use what I’ve learned from this course to write better-organized and concise HTML & CSS.
